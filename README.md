# timezone-hermes-fix

Fix for resetting the timezone cache, while the timezone is changed

## Installation

```sh
npm install timezone-hermes-fix
```

## Usage


```js
import { multiply } from 'timezone-hermes-fix';

// ...

const result = multiply(3, 7);
```
